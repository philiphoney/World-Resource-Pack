OptiFine is a mod that enables higher frame rates (FPS) in the game by optimizing the Minecraft code in many places. You can also make finer graphics settings (e.g. switching off individual particles). The mod also allows the player to zoom with the c key, which can be changed in the menu. It also adds features that many resource packs use and that can only be seen using OptiFine. OptiFine has also integrated a shader mod, with which shader effects can be used. Fullbright for Optifine also runs with this principle, unfortunately no shaders can be used by using the pack.
